The following table summarizes redeclaration, reassignment, scope, and hoisting behavior for let, var, and const:

Feature let var const

Function

A function is a block of code used to perform a specific task. It can accept input as parameters, execute instructions, and return a value. Functions enable code reusability. JavaScript supports both user-defined and built-in functions.

Function Declaration

Declared using the function keyword followed by a name, parentheses, and a block:

// Example (from the notes)
function Vedika() {
  console.log("My name is Vedika!");
}

Function Calling

Call/invoke a function using its name and parentheses:

Vedika();

What is Hoisting in JavaScript?

Hoisting is JavaScript's behavior of moving declarations (variables, functions, classes) to the top of their scope during the compilation phase. This can cause surprising behavior when using var, let, const, or function expressions.

Example from the notes:

a = 10;
console.log(a);
var a;

5
typeof

The typeof operator returns a string indicating the type of a variable or value.

  • Undefined — for an undefined variable 
    let a;
console.log(typeof a); // "undefined"
  • Boolean — for true or false 
    let d = false;
console.log(typeof d); // "boolean"
  • Number — for numeric values 
    let b = 10;
console.log(typeof b); // "number"
  • String — for text in quotes 
    let c = "Vedika";
console.log(typeof c); // "string"

When applied to a function, typeof returns "function".

Object in JavaScript

An object in JavaScript is a data structure used to store information in the form of key-value pairs. Each key is known as a property, and it can hold any value such as strings, numbers, arrays, functions, or even other objects.

Example: Creating an Object

      let student = {
        name: "Vedika",
        age: 20,
        isStudent: true,
        greet: function() {
          console.log("Hello!");
        }
      };

Accessing Properties

      console.log(student.name);      // Dot notation
      console.log(student["age"]);    // Bracket notation

Adding / Updating / Deleting Properties

      student.city = "Karad";   // Add
      student.age = 21;         // Update
      delete student.isStudent; // Delete

Objects with Methods

      let car = {
        brand: "Tata",
        start: function() {
          console.log("Car started");
        }
      };
      
      car.start();

Objects are widely used in JavaScript for modeling real-world entities, storing structured data, and building applications.

Functional Programming Language

Definition: A functional programming language is one in which functions are treated as first-class citizens. This means a function can:

  • Be returned from another function
  • Be stored in a variable
  • Be passed as an argument to another function

Example

      // Function stored in a variable
      let a = function(){
        console.log('I am a valid function');
        console.log('I am not a normal function');
        console.log('I am a anonymous  function');
      }
      
      // Function passed as an argument
      function dusrafunction(Kahitari){
        Kahitari();
        //return Kahitari;
      }
      dusrafunction(a);  //Prints statement from a

This shows that functions in JavaScript behave like values, which is a key feature of functional programming.

DOM (Document Object Model)

The DOM (Document Object Model) is a programming interface that represents a webpage as a tree structure of objects. Every element, attribute, and piece of text in an HTML document becomes a node in this tree. JavaScript uses the DOM to access, modify, add, or remove elements on a webpage.

Common DOM Methods

1. Selecting Elements

    document.getElementById("title");
    document.getElementsByClassName("box");
    document.getElementsByTagName("p");
    
    document.querySelector("#title");
    document.querySelectorAll(".box");

2. Changing HTML Content

    document.getElementById("title").innerHTML = "Welcome!";

3. Changing CSS Style

    document.getElementById("title").style.color = "red";

4. Changing Attributes

    document.getElementById("img").src = "photo.jpg";

5. Creating and Adding New Elements

    let newPara = document.createElement("p");
    newPara.textContent = "This is new!";
    document.body.appendChild(newPara);

6. Removing Elements

    document.getElementById("old").remove();

The DOM is essential for building interactive webpages because it allows JavaScript to dynamically modify content, styles, structure, and respond to user actions.
